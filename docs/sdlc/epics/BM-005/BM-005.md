# BM-005: Reading Practice

## Overview
Provides timed IELTS Reading practice sessions with multiple task types, auto-graded immediately on submission. No credits are consumed, making this a freely accessible practice module with detailed explanations surfaced after each attempt.

## Affected Modules

## Key Decisions
- Supported task types: Multiple Choice Questions (MCQ), True/False/Not Given, and Matching (headings, features, sentence endings)
- Sessions are timed to simulate exam conditions; the timer is enforced client-side with a server-side deadline check on submit
- Grading is fully automatic and synchronous; results are returned in the same HTTP response as the submission
- Reading practice does not consume credits
- Per-question explanations and the correct answer are shown immediately after submission

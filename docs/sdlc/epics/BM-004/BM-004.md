# BM-004: Speaking Practice

## Overview
Allows users to record and submit spoken responses to IELTS Speaking prompts for AI-powered scoring. Audio is uploaded to Supabase Storage, transcribed via a Whisper worker, and then scored across all four official speaking criteria, with credits deducted at submission.

## Affected Modules
speaking, submissions, scoring, storage, wallet

## Key Decisions
- User audio recordings are stored in Supabase Storage with a private bucket policy
- A dedicated Whisper transcription worker converts audio to text before the scoring pipeline runs
- Scoring uses the four official IELTS Speaking criteria: Fluency & Coherence / Lexical Resource / Grammatical Range & Accuracy / Pronunciation
- One credit is reserved from the user's wallet at the moment of submission
- Scoring pipeline is asynchronous via BullMQ, consistent with the Writing module approach

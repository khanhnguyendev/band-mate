# BM-009: Gamification System

## Overview
Introduces XP, streaks, badges, and a quest system to drive daily engagement and reward consistent study behaviour. Daily and weekly quests award bonus credits, and an opt-in leaderboard lets users compare progress with peers.

## Affected Modules
gamification, quests, wallet, notifications

## Key Decisions
- Core engagement mechanics: XP points (awarded per activity), login/practice streaks, and unlockable badges
- Daily and weekly quests have explicit completion conditions; completing a quest awards bonus credits to the wallet
- Leaderboard participation is opt-in; users are anonymous on the board by default until they choose a display name
- Mascot milestone reactions are triggered by key gamification events (level-up, streak milestone, badge unlock)
- Quest state and XP are tracked server-side to prevent client-side manipulation

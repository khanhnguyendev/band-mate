# BM-008: Dashboard and Progress

## Overview
Provides users with a centralised view of their learning journey, combining historical performance data, engagement streaks, and upcoming quest progress in a single dashboard. Visual components make progress tangible and motivate continued study.

## Affected Modules
dashboard, progress, gamification

## Key Decisions
- Band score trend chart visualises overall and per-skill score history over time
- Streak calendar shows daily activity, consistent with the gamification module definition of an active day
- Skill radar chart covers all four skills (Reading / Listening / Writing / Speaking) using the most recent scored submission per skill
- Recent submissions list shows the last N submissions with quick-access links to individual reports
- Quest progress widget surfaces active daily and weekly quests with completion percentage

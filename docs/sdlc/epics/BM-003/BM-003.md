# BM-003: Writing Practice

## Overview
Enables users to practise IELTS Writing Task 1 and Task 2 with AI-powered scoring across all four official criteria. Submissions are queued asynchronously via BullMQ to meet a 60-second scoring SLA, and credits are deducted at the point of submission.

## Affected Modules
writing, submissions, scoring, wallet

## Key Decisions
- Both Task 1 (graph/diagram description) and Task 2 (argumentative essay) are supported
- Scoring uses the four official IELTS criteria: Task Response / Coherence & Cohesion / Lexical Resource / Grammatical Range & Accuracy
- One credit is reserved from the user's wallet at the moment of submission; the deduction is finalised on scoring completion
- Scoring is processed asynchronously through a BullMQ queue on NestJS to avoid blocking the HTTP response
- Target SLA for score delivery is 60 seconds from submission

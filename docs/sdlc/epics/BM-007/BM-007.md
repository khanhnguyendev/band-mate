# BM-007: Reports and Improvement Plans

## Overview
Automatically generates a personalised performance report after each scored Writing or Speaking submission. Each report includes a band score breakdown, error analysis, and a two-week improvement plan, viewable in-app and exportable as a PDF.

## Affected Modules
reports, scoring, exercises

## Key Decisions
- A report is generated automatically once the scoring pipeline completes for a Writing or Speaking submission
- Report content includes: per-criterion band score breakdown, annotated error analysis, and a structured two-week improvement plan with linked exercises
- Reports are accessible as a web view within the app and can be exported to PDF
- The improvement plan links to specific Reading/Listening exercises to address identified weaknesses

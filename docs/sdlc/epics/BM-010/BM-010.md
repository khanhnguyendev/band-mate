# BM-010: Mascot and UI Delight

## Overview
Defines the visual personality and motion design of Band Mate, centred on a mascot character that reacts to key user moments. Polished UI touches — including Lottie animations, confetti, dark mode, and smooth transitions — differentiate the product from clinical test-prep tools.

## Affected Modules

## Key Decisions
- Mascot animations are implemented with Lottie (JSON-based) for scalable, performant playback on web
- Confetti burst is triggered on meaningful achievements (badge unlock, level-up, streak milestone)
- Dark mode is a first-class UI requirement, not a post-launch addition; uses CSS custom properties / Tailwind dark variant
- Page transitions use smooth animation (e.g. Framer Motion) to reinforce the app's polished feel
- Mascot reacts contextually to: submission complete, streak maintained, streak broken, level-up, and low-credit warning

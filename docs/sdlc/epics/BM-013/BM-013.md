# BM-013: Admin Console

## Overview
Provides an internal-only operations interface for the Band Mate team to manage users, monitor the scoring pipeline, adjust credits, version prompt packs, moderate content, and view revenue metrics. Access is restricted to accounts with the admin role.

## Affected Modules
admin, users, scoring, wallet, content

## Key Decisions
- All admin routes are protected by a `role=admin` guard; the console is not accessible to regular users
- User management features: search, view profile, suspend/unsuspend account, and force-reset password
- Manual credit adjustment is supported with a mandatory reason field that is recorded in the ledger
- Scoring queue monitor displays queue depth, processing rate, error rate, and allows manual retry of failed jobs
- Prompt pack versioning allows new AI scoring prompts to be staged, reviewed, and promoted without a code deploy
- Content moderation flags surface user-reported submissions for review and action (approve/remove)
- Revenue metrics view includes MRR, churn, credit consumption rate, and per-tier subscriber counts

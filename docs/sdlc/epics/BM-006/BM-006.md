# BM-006: Listening Practice

## Overview
Delivers IELTS Listening practice using audio served from Supabase Storage, covering all four sections of the academic exam format. Multiple task types are supported, grading is automatic, and no credits are required to practise.

## Affected Modules
listening, exercises, storage, progress

## Key Decisions
- Audio files for exercises are stored in and served directly from Supabase Storage
- Exercise content follows the IELTS Listening Sections 1–4 format (social/transactional, monologue, discussion, academic lecture)
- Supported task types: form completion, MCQ, and map/diagram labelling
- Listening practice does not consume credits
- Grading is automatic and synchronous, consistent with the Reading module approach

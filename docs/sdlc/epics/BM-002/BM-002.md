# BM-002: Onboarding

## Overview
Guides new users through a post-registration setup flow to personalise their learning experience. Collects self-assessed English level, weekly study goal, and target band score, then recommends an appropriate subscription plan tier based on those inputs.

## Affected Modules
onboarding, users, plans

## Key Decisions
- Onboarding flow is triggered immediately after successful registration and email verification
- Three data points collected: current English level, weekly study goal (hours/sessions), and target band score
- Plan tier recommendation is generated from collected inputs and displayed at the end of the flow
- Users may skip individual steps; skipped fields default to sensible values and can be updated later in settings

# Module 9: Gamification system

## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
XP for broad learning activity. Credits earnable from low-cost validated actions (Reading, Listening, report review, drills). Daily quests, weekly challenges, streaks, badges, milestones. Daily and weekly earn caps. Anti-abuse rules.
<!-- SECTION:DESCRIPTION:END -->

## Acceptance Criteria
<!-- AC:BEGIN -->
- [ ] #1 User sees daily quests and weekly challenge with progress
- [ ] #2 Daily and weekly credit earning caps are enforced
- [ ] #3 Only validated completions trigger rewards
- [ ] #4 Admin can change reward values without redeploying
<!-- AC:END -->
